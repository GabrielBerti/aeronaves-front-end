ESTE PROJETO CONSISTE NA PARTE FRONT-END DO PROJETO DE "GESTÃO DE AERONAVES"

-- É um projeto que utiliza as tecnologias Angular e Bootstrap 
-- Para abrir o projeto e codificar é necessário instalar o Visual Studio Code
-- Para executar o projeto é necessário estar com o projeto back-end em execução e também é necessário instalação do Angular CLI (link para tutorial de instalação: https://www.devmedia.com.br/angular-cli-instalacao/38247)
-- Após instalado o Angular CLI é necessário ir até a pasta do projeto via CMD e executar o comando: ng serve --open
-- Após o servidor ter iniciado a aplicação já estará disponível  para ser acessado pela URL: http//localhost:4200
